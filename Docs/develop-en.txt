PAP system design, catalog module structure
B/S mode and C/S mode , the operating environment is: lnmp + lua
Involved languages: C/C++ LUA PHP SHELL
PHP framework: zendframework 2 Server architecture: self-developed

Main directory structure ( ./ )
.
|-- composer.json -- PHP framework JSON
|-- composer.phar -- PHP framework Upgrade Script
|-- config -- PHP framework configuration directory
|-- data -- PHP framework Data Directory
|-- Docs -- Documentation directory

|-- LICENSE.txt
|-- module -- PHP framework Module Directory
|-- public -- PHP framework Public Directory
|-- README.md
